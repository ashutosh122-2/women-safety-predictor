# 🚨 Women Safety Predictor Web App

## 📌 Overview
The **Women Safety Predictor Web Application** is a machine learning-based system designed to predict the safety level of a particular location using different parameters.  
The application analyzes input data using a trained machine learning model and predicts whether an area is **safe or potentially risky**.

The main objective of this project is to combine **Machine Learning and Web Development** to create a system that can help analyze safety conditions using data-driven predictions.

---

## 🛠 Technologies Used

### 1️⃣ Python
Python is the primary programming language used in this project.

It is used for:
- Data preprocessing
- Machine learning model development
- Backend logic
- Prediction processing

Python is widely used in machine learning because of its powerful ecosystem and libraries.

---

### 2️⃣ Flask
Flask is a lightweight **Python web framework** used to develop the backend of the web application.

Flask handles:
- User requests
- Communication between the web interface and machine learning model
- Data processing
- Returning prediction results to the user interface

---

### 3️⃣ Machine Learning
Machine Learning is used to analyze input parameters and predict the **safety level of a location**.

The ML workflow includes:
- Data preprocessing
- Feature selection
- Model training
- Model evaluation
- Generating predictions

---

### 4️⃣ Scikit-learn
Scikit-learn is a powerful Python library used for building machine learning models.

It is used for:
- Model training
- Dataset splitting (training and testing)
- Model evaluation
- Prediction generation

---

### 5️⃣ Pandas
Pandas is used for **data manipulation and data analysis**.

It helps in:
- Reading datasets
- Cleaning and preprocessing data
- Managing structured tabular data
- Preparing data for machine learning training

---

### 6️⃣ NumPy
NumPy is used for numerical computations.

It helps in:
- Mathematical calculations
- Handling numerical datasets
- Supporting machine learning operations

---

### 7️⃣ HTML
HTML is used to create the **structure of the web interface**.

It provides:
- Input forms
- Page layout
- Display of prediction results

---

### 8️⃣ CSS
CSS is used to improve the **visual design of the web application**.

It helps with:
- Styling the interface

---

## ✨ Features
- Machine Learning based safety prediction

---

## 🏗 Project Architecture

1. User enters input data through the web interface  
2. Flask server receives the request  
3. Input data is sent to the trained machine learning model  
4. The model processes the data and generates a prediction  
5. The prediction result is displayed on the web page
